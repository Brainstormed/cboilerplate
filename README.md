# C BOILERPLATE GENERATOR

This simple utility can be used to create a very minimal C Boilerplate.

## How to use

### Clone git repo

```$ git clone https://github.com/Brainstormed/cboilerplate.git```

```$ cd cboilerplate```

### Build the project

```$ clang main.c -o cb```

### Move to user binaries directory (Optional)

```$ sudo mv cb /usr/bin```
